
# 📚 StudyMate - UI/UX Design (Task 1)

This project is my **Task 1** submission for the **CodSoft UI/UX Design Internship**.

## 📱 Project Overview

**StudyMate** is a clean and modern mobile app UI designed to help students manage their learning journey. The interface focuses on simplicity, usability, and an attractive visual experience.

## 🎨 Screens Included

- 🏠 Welcome Screen
- 📝 Sign Up Screen
- 🔐 Login Screen
- ✅ Welcome Back Popup
- 🎉 Account Created Success Popup

## ✨ Design Features

- Modern and minimal UI
- Soft pink color palette
- User-friendly navigation
- Clean typography
- Consistent spacing and layout
- Mobile-first design approach

## 🛠️ Design Tool

- Figma

## 🎯 Objective

The goal of this project was to design an intuitive and visually appealing user interface for a student learning application while following basic UI/UX design principles.
