
# 🍣 SushiZen - Food Ordering Mobile App UI (Task 3)

This project is my **Task 3** submission for the **CodSoft UI/UX Design Internship**.

## 📱 Project Overview

**SushiZen** is a modern and user-friendly mobile food ordering application designed for sushi lovers. The interface focuses on providing a smooth browsing and ordering experience with an attractive visual design and intuitive navigation.

## 🎨 Screens Included

- 🍣 Welcome Screen
- 🏠 Home Screen
- 🍱 Product Details Screen
- ✅ Add to Cart Confirmation Popup

## ✨ Design Features

- Modern and clean UI
- Japanese-inspired color palette
- Easy navigation
- Category-based menu browsing
- Search and filter functionality
- Product ratings and pricing
- Quantity selector
- Attractive Call-to-Action (CTA) buttons
- Mobile-first responsive design

## 🛠️ Design Tool

- Figma

## 🎯 Objective

The objective of this project was to design an intuitive and visually appealing mobile food ordering application that provides users with a seamless experience for discovering, exploring, and ordering their favorite sushi dishes.

## 📚 UI/UX Principles Applied

- User-Centered Design
- Visual Hierarchy
- Consistent Color Scheme
- Simple Navigation
- Accessibility
- Responsive Mobile Layout
