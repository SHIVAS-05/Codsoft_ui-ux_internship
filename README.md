# 📂 Projects

## 📚 Task 1 – StudyMate (Learning Management Mobile App)

### 📌 Description

StudyMate is a mobile application designed to help students manage their learning journey efficiently. The interface provides a clean and simple experience for new and existing users through attractive onboarding, registration, and login screens.

### ✨ Features

- Welcome Screen
- Sign Up Screen
- Login Screen
- Welcome Popup
- Account Created Success Popup
- Clean and Modern UI
- Mobile-first Design
- Easy Navigation

---

## 📧 Task 2 – SkillBoost Academy (Email Newsletter)

### 📌 Description

SkillBoost Academy is a professional educational email newsletter created to promote online learning courses, career resources, and skill development opportunities. The newsletter is designed with a clean layout, engaging visuals, and clear call-to-action buttons to maximize user engagement.

### ✨ Features

- Professional Email Layout
- Featured Learning Section
- Popular Courses
- Career Resources
- Call-to-Action Buttons
- Social Media Integration
- Responsive Design
- Modern Typography

---

## 🍣 Task 3 – SushiZen (Food Ordering Mobile App)

### 📌 Description

SushiZen is a modern food ordering mobile application designed for sushi lovers. The application offers a seamless experience for browsing food items, viewing product details, selecting quantities, and placing orders through a visually appealing interface.

### ✨ Features

- Welcome Screen
- Home Screen
- Search Functionality
- Food Categories
- Product Details
- Quantity Selector
- Add to Cart Interaction
- Success Confirmation Popup
- Responsive Mobile Layout
