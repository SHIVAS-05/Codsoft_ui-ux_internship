
# 📧 SkillBoost Academy - Email Newsletter Design (Task 2)

This project is my **Task 2** submission for the **CodSoft UI/UX Design Internship**.

## 📌 Project Overview

**SkillBoost Academy** is a modern and visually engaging email newsletter designed to promote online learning courses, career resources, and educational content. The design focuses on readability, user engagement, and clear call-to-action elements.

## 🎨 Design Highlights

- Modern and clean layout
- Attractive green-themed color palette
- Easy-to-read typography
- Responsive email newsletter structure
- Clear call-to-action (CTA) buttons
- Organized course categories
- Social media integration
- Professional footer with unsubscribe option

## 📋 Sections Included

- Header with branding
- Featured learning section
- Popular courses
- Career resources
- Promotional banner
- Social media links
- Footer

## 🛠️ Design Tool

- Figma

## 🎯 Objective

The objective of this project was to design a user-friendly and visually appealing email newsletter that encourages learners to explore online courses while maintaining a clean and professional layout.
